# Rogue Clone II UI (Rogue Clone II for Web)

名作ローグライクゲーム「Rogue Clone II」を、モダンなUIかつ現代のWebブラウザで遊べるように移植・再構築した趣味のプロジェクトです。

[🎮 **Play Game here!**](https://yosssie.github.io/RogueC2UI/)

## 📜 About This Project

このプロジェクトは、**スズキイイチロウ氏**による「Rogue Clone II」のオリジナルソースコードを最大限に尊重し、参照させていただきながら開発されました。
オリジナル版を作成・公開してくださったことに深く感謝いたします。

- **Original Reference**: [suzukiiichiro/Rogue2.Official](https://github.com/suzukiiichiro/Rogue2.Official)

オリジナルのC言語コードのロジックを読み解き、忠実に再現することを目指しています。

## 🤝 Developed with Vibe Coding

このゲームは、生成AIとの対話を通じた **"Vibe Coding" (バイブコーディング)** によって開発されています。
オリジナルのロジックを一つ一つAIと解析し、JavaScriptの世界へと移植を行いました。
古き良きアルゴリズムと、現代のAIコーディングの融合をお楽しみください。

## 🕹️ Controls (操作方法)

PCのキーボードで、スーパーファミコンのような操作感で遊べるように調整されています。
（viキーバインドは現在無効化されています）

### 基本操作
- **移動**: `矢印キー` または `テンキー` (8方向)
- **決定**: `Z` または `Enter`
- **キャンセル**: `X` または `Escape`

### アクションキー
- **`Z`**: 足踏み (休息しながら周囲を探索)
- **`X`**: ダッシュ (押しながら移動キーで高速移動)
- **`A`**: メニューを開く
- **`S`**: 向きを変える(未実装)
- **`Q`**: 矢を放つ(未実装)
- **`W`**: 斜め移動固定 (押しながら移動キーで斜めのみ入力(未実装))

### メニュー内操作
- **`矢印キー`**: カーソル移動
- **`Z` / `Enter`**: 決定
- **`X` / `Escape`**: 戻る / 閉じる
- **`S`**: 整頓

### その他
- **`D`**: デバッグダンジョン (タイトル画面のみ)
- **`D`**: デバッグ表示 (ゲーム中)

> ※オリジナルのRogueのような細かいコマンド入力（e, q, r, w...）は使えないようにしてあります。
