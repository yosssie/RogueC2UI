# Rogue Web (Rogue Clone II for Web)

名作ローグライクゲーム「Rogue Clone II」を、現代のWebブラウザで遊べるように移植・再構築した趣味のプロジェクトです。

[🎮 **Play Game here!**](https://yosssie.github.io/RogueC2UI/)

## 📜 About This Project

このプロジェクトは、**スズキイイチロウ氏**による「Rogue Clone II」のオリジナルソースコードを最大限に尊重し、参照させていただきながら開発されました。
偉大なオリジナル版を作成・公開してくださったことに深く感謝いたします。

- **Original Reference**: [suzukiiichiro/Rogue2.Official](https://github.com/suzukiiichiro/Rogue2.Official)

オリジナルのC言語コード (`move.c`, `fight.c`, `monster.c` 等) のロジックを読み解き、忠実に再現することを目指しています。

## 🤝 Developed with Vibe Coding

このゲームは、生成AIとの対話を通じた **"Vibe Coding" (バイブコーディング)** によって開発されています。
オリジナルのロジックを一つ一つAIと解析し、JavaScriptの世界へと移植を行いました。
古き良きアルゴリズムと、現代のAIコーディングの融合をお楽しみください。

## 🕹️ Controls (操作方法)

基本的にオリジナルRogue準拠（Viキーバインド対応）です。
モバイル対応のUIも備えています。

### キーボード操作
- **移動**: `h`, `j`, `k`, `l`, `y`, `u`, `b`, `n` / テンキー / 十字キー / WASD
- **ダッシュ**: `Shift` + 移動キー
- **キャンセル**: `ESC`
- **決定/進む**: `Enter`

### アクション
- **`s`**: 探索 (Search)
- **`>`**: 降りる
- **`i`**: 持ち物 (Inventory)
- **`e`**: 食べる (Eat)
- **`q`**: 飲む (Quaff potion)
- **`r`**: 読む (Read scroll)
- **`w`**: 武器を装備 (Wield weapon)
- **`W`**: 防具を着る (Wear armor)
- **`T`**: 防具を脱ぐ (Take off armor)
- **`d`**: 落とす (Drop)
- **`t`**: 投げる (Throw)
- **`z`**: 杖を振る (Zap wand)
- **`P`**: 指輪をはめる (Put on ring)
- **`R`**: 指輪を外す (Remove ring)
